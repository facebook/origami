Scroll

Lets you scroll a layer using free, paging, or "wheel of fortune" scrolling.

To use:

First connect the Image/Layer Group you want to scroll to the Scroll and Layer patches so the Scroll patch knows the dimensions of what it's scrolling and the image is displayed.

Then connect the Scroll patch’s X and/or Y positions to the Layer’s X and/or Y positions.

If using multiple Scroll patches, tie each one to a specific layer by connecting the Scroll patch's interaction port (link) on the top right to the Layer's interaction port on the top left. Note: if you are scrolling a Layer Group, layers inside the Layer Group may intercept touch events (link to Interaction patch events section).

Enable

Turn off to stop touches from scrolling.

Image

The image you want to scroll. The patch uses this to determine the size of what you're scrolling. This image should be the same size as the layer that you intend to scroll. This is optional if you specify Content Width/Height in an attached Scroll Settings patch.

Note: The image needs to be wider (if scrolling horizontally) or taller (if scrolling vertically) than the containing Layer Group for it to scroll.

X Scrolling

Sets the horizontal scrolling behavior to Free (default), Paging (e.g. iOS home screen), or Wheel of Fortune (paging with the ability to flick harder to scroll through more pages, e.g. cover flow).

Y Scrolling

Sets the vertical scrolling behavior to Free, Paging, or Wheel of Fortune.

Settings

Takes optional settings from a Scroll Settings or Scroll Advanced Settings patch.

X Position

The horizontal position of the scroller. Connect to the X Position of the Layer you want to scroll horizontally.

Y Position

The vertical position of the scroller. Connect to the Y Position of the Layer you want to scroll vertically.

Info

Additional information about the current Scroll status. Connect to a Scroll Info patch to read it.
